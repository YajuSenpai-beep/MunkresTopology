## 当前项目能力
- Python 包：16 个模块（engine, parser, tex_utils, matcher, config, project, scanner, cli, tools_cli, collation, reporter, tui, latexmk, xindy 等）
- 测试：409 项 pytest，覆盖率 89%，含模糊测试
- 安全：原子写入、文件锁、备份回滚、编码检测、换行保留、日志轮转
- 性能：自适应策略（逐条/ Aho-Corasick），大文件缓冲，进度条
- CLI：insert, parse, scan, report, setup, rollback, xindy, tools 共 8 个子命令
- 集成：Rich TUI 交互、latexmk 编译前置、xindy 排序规则生成
- 文档：完整的 README、配置手册、CLI 参考、安全特性、贡献指南
- CI/CD：GitHub Actions 多版本测试 + lint + mypy + 性能基准 + 自动发布 PyPI 和 Docker
