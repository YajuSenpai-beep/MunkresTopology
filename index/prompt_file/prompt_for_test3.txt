## 当前项目概览
这是一个名为 `latex-index-tool` 的通用 LaTeX 索引插入工具，项目结构如下：
- Python 核心包 `latex_index/`（包含 engine、parser、tex_utils、matcher、config、project、scanner、cli、tools_cli 等模块）
- 配置驱动（YAML），支持别名、数学快捷方式、多模板
- 安全特性：原子写入、dry-run、交互确认、LaTeX 语义安全跳过（数学/注释/verbatim/命令参数等）
- 性能：自适应策略（常规逐条搜索，大规模自动切换 Aho-Corasick 自动机，大文件流式处理）
- 测试：82 项 pytest 测试（单元/集成/回归/异常/性能），搭配 ruff、mypy strict 类型检查
- CLI：支持子命令 insert、parse、scan、tools，提供进度条、编码自动检测、换行符保留等
- 文档：README 已有快速开始、CLI 参考、配置手册、安全特性、性能说明等
